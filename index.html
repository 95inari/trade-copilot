J-Quants連携

確認中…

日本取引所グループ公式のデータAPIです。SNSログイン後に発行する V2 APIキーで連携すると、 銘柄の正式名称・業種・休場日の自動判定・参考統計が使えるようになります。 無料プランでも利用できます。ブラウザの制約があるため、先に下の データ中継プロキシURLを設定してください。

  1. J-Quants公式サイト にSNSでログインし、無料プランを選ぶ
  2. ダッシュボードでV2 APIキーを発行してコピーする
  3. APIキーを下に貼り付ける
  4. 「接続して保存」を押す

APIキーはこの端末のブラウザ内だけに保存され、接続時は自分専用の中継を経由してJ-Quants公式APIへ送信されます。 中継には保存しません。 共有端末では使用しないでください。

データ設定

GitHub Pages版はブラウザの制約（CORS）でJ-QuantsとYahoo Financeへ直接アクセスできないため、 自分専用の中継プロキシ（Cloudflare Worker・無料）が必要です。 リポジトリの proxy/cloudflare-worker.js を約5分でデプロイし、URLをここに貼り付けてください。 未設定時や取得失敗時は、誤認防止のため銘柄・価格・スコアを表示しません。

日誌・取引・設定はこの端末のブラウザ内にのみ保存されます。 iOSのSafariは7日間使わないとデータを自動削除することがあります。 また「ホーム画面に追加」したアプリ版とブラウザ版はデータが別々です。 大事な記録は定期的にバックアップ保存してください（J-Quantsの認証情報は含まれません）。

本アプリは投資判断の補助・記録ツールであり、投資助言・売買推奨ではありません。実注文機能はなく、ペーパートレードのみです。 表示データには遅延・誤差があり得ます。投資判断はご自身の責任で行ってください。
