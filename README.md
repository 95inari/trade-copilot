# Trade Co-Pilot（PWA版）

日本株デイトレードの監視・リスク管理・取引日誌ツールです。ブラウザだけで動作し、スマホの「ホーム画面に追加」でアプリとして使えます。

公開版はCloudflare Workerのサーバー側パスワード認証で保護し、認証済みの利用者にだけPWAファイルを配信します。

> **免責**: 本アプリは投資判断の補助・記録ツールであり、投資助言・売買推奨ではありません。実注文機能はなく、ペーパートレード（仮想売買の記録）のみです。表示データには遅延・誤差があり得ます。投資判断はご自身の責任で行ってください。

## 主な機能

- **朝トレルール**: 値動きの大きい朝（9:05〜11:30）だけ新規を取り、昼以降は追わない運用をデフォルト化。11:30終了と利確・損切り到達を通知
- **市場全体スキャン**: 値上がり率・出来高増加率・売買代金ランキングから最大18銘柄を自動発掘し、固定監視銘柄と合わせて比較
- **リスク管理**: 1回/1日の損失上限・連敗停止・損切り未設定警告・ロット計算（リスクリワード表示付き）
- **ペーパー自動売買**: 条件一致銘柄の仮想購入・損切り/利確の自動決済・成績集計（勝率・平均損益・最大DD・戦略別）
- **取引日誌**: ルール違反の自動記録、日次・週次集計、反省コメント生成
- **J-Quants連携**: SNSログイン後に発行したV2 APIキーで、銘柄の正式名称・業種・営業日カレンダー・参考統計を取得。公式の上限に合わせて通信間隔を自動調整

## データについて

- 記録はすべて**この端末のブラウザ内**に保存されます（サーバーには何も送信されません）
- **バックアップ推奨**: iOSのSafariは7日間未使用でデータを削除することがあります。「データ設定」からエクスポートできます
- **初期状態では価格非表示**: 実データ未接続時や取得失敗時は、誤認を防ぐため銘柄・価格・スコアを表示しません
- **データ中継プロキシ**: ブラウザの制約（CORS）のため、J-Quants V2とYahoo Financeの実データには自分専用の中継プロキシが必要です。`proxy/cloudflare-worker.js` をCloudflare Workers（無料）にデプロイし、URLをアプリの「データ設定」に貼り付けてください
  1. https://dash.cloudflare.com/ で無料アカウント作成
  2. Workers & Pages → Create Worker → `proxy/cloudflare-worker.js` の内容を貼り付けて Deploy
  3. 表示されたURL（`https://xxx.workers.dev`）をアプリの「データ設定」に入力
- **J-QuantsのV2 APIキーは端末内だけに保存**され、接続時は自分専用の中継を経由してJ-Quants公式APIへ送信されます。中継には保存しません

## 開発

ローカル版（FastAPI・実データ取得サーバー付き）を含む開発リポジトリは非公開です。このリポジトリはPWA配布用です。

ローカルで動かす:

```bash
python3 -m http.server 8090
# → http://localhost:8090
```
